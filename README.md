# Cybersecurity & Digital Forensics Portfolio

This repository documents my 100 Days of Cybersecurity Challenge.

## What this includes:
- Hands-on practice using Kali Linux
- Tools: Nmap, Whois, Strings
- Basic malware analysis and network scanning
- Screenshots and explanation files

## Goal:
To build practical skills in cybersecurity and digital forensics.

## Structure:
Each folder contains:
- Screenshots
- Explanation (Objective, Procedure, Observation, Conclusion, Investigation Insight)
